新企画、陽月通信始動！

新しく陽月通信を始めました。

陽月通信というウェブサイトを製作いたしました。これを製作したきっかけは自分の趣味を出したいというところと ウェブサイトの制作をやってみたいと思ったからです。どうやって作ったのか疑問に思う人もいるかもしれませんが、 今回はhtmlとcssとjavascriptという自分もよくわかっていない謎技術を使用し、やってみました。
  • HTMLとは「Hyper Text Markup Language」の略で、ウェブページの構造を定義する言語です。文字、画像、動画などのコンテンツを配置する際に使用します。HTMLのタグを使って、見出し、段落、リンクなどを作成します。
  • CSSとは「Cascading Style Sheets」の略で、HTMLで定義した要素の見た目を設定する言語です。文字の色や大きさ、背景色、レイアウトなどを自由にデザインできます。HTMLとCSSを組み合わせることで、見栄えの良いウェブページが作れます。
  • JavaScriptは、ウェブページに対話性や動きを付加する scripting 言語です。ボタンをクリックしたら何かが起こる、マウスを動かすと画面が反応するなど、ユーザーとのインタラクションを実現できます。HTMLとCSSでは表現できない動的な挙動を追加することができます。
このような自分でも全然わからない技術を使っていますが、ウェブサイトを作っている人にはReactというゲームでいう拡張機能みたいなのを使ったり、Next.jsという噂Reactにさらにドッキングすることででは裏方から表側まで やってくれるという嬉しい機能が付いているが、ダウンロード方法すら（私の能力的に）わからない技術、TypeScriptやbootstrapのJavaScriptやCSSの派生形と思われるなんか。などと説明文を見てわかる通り 自分にはまだ理解できていない分野で作っている方もいるそうです。（実際ダンマクカグラ公式のウェブサイトはReactやNext.jsと同じような、ウェブサイト制作をサポートするVue.jsやNuxt.jsが使われている。） このように勉強し始めて４か月くらいしか立っていないし、それにしてもまるで成長していない気がしますが、これからも更新する予定なので宜しくお願い致します。
